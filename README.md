# Application Desktop de Gestion de Commerçant

Cette application est une application desktop développée en JavaFX avec une base de données MySQL (JDBC).

## Objectif
L'objectif de cette application est de permettre la gestion d’un commerçant, notamment :
- La gestion des produits
- La gestion des clients
- La gestion des ventes
- La gestion du stock

##  Technologies utilisées
- Java
- JavaFX (Interface graphique)
- MySQL
- JDBC

##  Fonctionnalités principales
- Ajouter, modifier, supprimer des produits
- Ajouter, modifier, supprimer des clients
- Enregistrer des ventes
- Mise à jour automatique du stock
- Connexion à la base de données MySQL
